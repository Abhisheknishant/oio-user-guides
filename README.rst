Requirements
------------

Tested successfully with:

- docker-compose v1.25.1 (version 1.22 required at least: https://github.com/docker/compose/blob/master/CHANGELOG.md#compose-format-version-24)
- docker 18.09.1
- GNU Make 3.81
- Bats v0.4.0
